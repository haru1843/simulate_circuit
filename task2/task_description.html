本課題では, JKフリップフロップを用いた16進カウンタを構成する.
16進カウンタとは, クロックパルスを入力することで, 4ビットの数値が 1 ずつ増えていき, 1111(2)の時に, 0000(2)に戻るということを繰り返すような順序回路のことである.
回路構成の流れ
  1. 4ビット16進カウンタの状態遷移表を作成する
  2. 状態遷移表をもとに, J, Kについての論理式を導出する (カルノー図等を用いる)
  3. 導出された論理式をもとに, JKフリップフロップを用いた回路を構成する
確認事項
  1. クロックスイッチを押すごとに, ディスプレイ上の数値が進む
  2. ディスプレイ上の数値が F の時にクロックスイッチを押すと, 0 に戻る
  3. (動作の詳細は以下のアニメーションを参照)
注意事項
  • クロックスイッチによる入力のみで, カウンタが進むような回路構成にすること
  • JKフリップフロップの特性をよく考えて, 回路を構成すること
