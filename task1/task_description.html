本課題では, 3bit の 2進数を入力として, 7セグメントディスプレイ上に, 0 ~ 7を表示する回路を構成する.
回路構成の流れ
  1. 入力する 3bit の 2進数と, 各セグメントにおける真理値表を作成する
    (例 : 入力が, 100(2) の場合, 7セグメントディスプレイの 2, 3, 6, 7入力目に1を入力する)
  2. 1 で作成した真理値表をもとに, 各セグメントについてのカルノー図を作成し, 簡略化した論理式を導出する
  3. 導出した論理式をもとに回路を構成する
確認事項
  1. トグルスイッチを切り替えて入力を変えることで, それに対応した値がディスプレイ上に表示されている
注意事項
  • 既に構成されているデバイスを基本とし, スイッチとディスプレイ間の回路を考えること
  • 回路構成をわかりやすくするために, 論理式の簡略化や, デバイスの配置を工夫すること
